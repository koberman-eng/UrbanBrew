# Анимации для Urban Brew

## Описание

Эта папка содержит файлы анимаций для веб-сайта кофейни Urban Brew.

## Файлы

### 1. logo-animation.html
CSS анимация логотипа с вращающейся чашкой и паром.

**Использование:**
- Откройте в браузере для просмотра
- Используйте как референс для создания GIF
- Можно интегрировать напрямую в HTML (CSS анимация)

**Экспорт в GIF:**
1. Запишите экран с помощью инструмента записи
2. Импортируйте в Photoshop
3. Экспортируйте как GIF (см. EXPORT_INSTRUCTIONS.md)

### 2. hero-banner-animation.html
CSS анимация баннера с появлением элементов и плавающими объектами.

**Использование:**
- Откройте в браузере для просмотра
- Используйте как референс для создания видео
- Можно интегрировать напрямую в HTML (CSS анимация)

**Экспорт в MP4:**
1. Запишите экран с помощью инструмента записи
2. Оптимизируйте видео (см. EXPORT_INSTRUCTIONS.md)
3. Сохраните как MP4

**Экспорт в Lottie:**
1. Создайте анимацию в After Effects на основе HTML
2. Экспортируйте через плагин Bodymovin
3. Сохраните как JSON

## Рекомендации

- Для веб-использования предпочтительнее CSS анимации (меньший размер)
- Для сложных анимаций используйте Lottie (векторная, масштабируемая)
- Видео используйте только для очень сложных анимаций
- Всегда оптимизируйте размер файлов

## Интеграция

См. `demo-page.html` для примеров интеграции всех типов анимаций.
